bitzeny_miner
====

This is dockerized bitzeny miner.

## Description
dockerized simple bitzeny miner

## Requirement
docker  
docker-compose

## Usage
open .env  
add pool url  
add username  
add password

```
docker-compose up
```

## Example
edit .env  
pool_url is your pool url  
username is your pool username  
password is your pool password
```
POOL_URL=pool_url
USERNAME=username  
PASSWORD=password
```
